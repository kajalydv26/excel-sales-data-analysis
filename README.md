# Sales Data Analysis using Microsoft Excel

## Project Overview

This project analyzes a retail sales dataset using Microsoft Excel to identify sales trends and business insights.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Filtering

## Key Analysis

* Analyzed sales performance across different regions
* Identified top-performing products and categories
* Examined monthly sales trends

## Dashboard

Created an interactive dashboard in Excel to visualize:

* Sales by Region
* Profit by Category
* Monthly Sales Trends
